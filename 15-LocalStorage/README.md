# 15 - LocalStorage

## 目標

- 了解如何透過瀏覽器的 LocalStorage 來儲存資料 (防止 refresh、close 導致資料消失)。

## 筆記

- 瀏覽器的儲存空間可分為 LocalStorage、SessionStorage 等。

- `JSON.stringify()` 將資料轉成 JSON；`JSON.parse()` 將 JSON 轉成原來的資料、型別。
  
- HTML 標籤的 `data-*` 屬性所擁有的值都會放在 `HTMLElement.dataset` 內。
