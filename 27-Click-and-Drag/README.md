# 27 - Click and Drag

## 目標

- 實作可向左、右拖曳的目錄。

## 筆記

- 使用 `scrollLeft` 取得元素在 X 軸上的捲動量、使用 `clientX` 取得 X 軸與瀏覽器窗口的距離。
