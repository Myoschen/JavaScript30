# 14 - JavaScript References VS Copying

## 目標

- 了解 JavaScript 的資料型別。

## 筆記

- `string`、`number`、`boolean`、`null`、`undefined`、`symbol` 都屬於簡單資料型別；`object`、`array`、`function` 則屬於複雜資料型別。

- 了解 by value、by reference 差異。
