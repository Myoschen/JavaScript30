# 20 - Speech Detection

## 目標

- 使用 SpeechRecognition 來進行語音識別。

## 筆記

- 使用 `SpeechRecognition.start()` 啟動語音識別。
- 監聽 `result` 來取得識別結果、透過 `isFinal` 確認是否為最終結果。
