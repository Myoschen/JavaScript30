# 08 - Fun with HTML5 Canvas

## 目標

- 此次為練習操作 Html5 的畫布（`canvas`）功能。

## 筆記

- 了解 `canvas` 標籤作用、及如何透過 JavaScript 操作。

    ```js
    // 取得畫布
    const canvas = document.querySelector('#draw');
    const ctx = canvas.getContext('2d');
    ```
  
- 了解 HSL 色彩碼。
