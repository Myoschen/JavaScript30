## 筆記

- `this` 會取得被監聽的對象本身；而 `e.target` 則是會取得觸發事件時的對象。
- `offsetWidth`、`offsetHeight` 是 `Element` 的屬性，表示該 element 的寬高；而 `offsetX`、`offsetY` 是 `Event` 的屬性，表示滑鼠至外層容器的距離。
- `offsetParent` 是與元素最近的，有被定位 (position 不等於 static) 的父層。
