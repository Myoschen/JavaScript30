# JavaScript30

## 介紹

這是由 WES BOS 所製作的一系列純 JavaScript 練習。
總共有 30 個主題練習，其中每個主題都有教學影片及範例檔案可參考。

- 範例檔案：[WES BOS's Github](https://github.com/wesbos/JavaScript30)
- 課程官網：[JavaScript30 Official website](https://javascript30.com/)
