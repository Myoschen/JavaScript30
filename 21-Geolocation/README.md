# 21 - Geolocation

## 目標

- 使用 `Geolocation API` 取得使用者位置資訊。

## 筆記

- `navigator.geolocation.watchPosition()` 追蹤當前位置，回傳 `GeolocationPosition` 物件包含了經度、緯度、設備速度、方向等資訊。
