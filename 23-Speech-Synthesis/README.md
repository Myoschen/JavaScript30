# 23 - Speech Synthesis

## 目標

- 使用 `SpeechSynthesis API` 將輸入文字轉為語音。

## 筆記

- 透過 `SpeechSynthesisUtterance()` 設定語言、音調、聲音、速率，以及要講的文字。
- `speechSynthesis.speak(msg)` 撥放發音、`speechSynthesis.cancel()` 移除發音。
